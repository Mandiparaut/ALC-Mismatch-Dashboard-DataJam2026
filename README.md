# ALC Mismatch Dashboard — DataJam 2026
**Top 6 Finalist — NS Healthcare Track**
SMU Sobey School of Business · March 7, 2026

## The Problem
545 Nova Scotia hospital beds are blocked every day 
by patients medically ready to leave but with nowhere to go.

## Our Solution
A dashboard using real NS government data that shows 
where the mismatch is worst and models the impact of 
redirecting existing home care resources.

## What We Found
- Central NS has the worst ALC rate — 23.1%
- 218 beds blocked daily in Halifax alone
- Redirecting 30% of home care saves $69M/year
- No new buildings. No new staff. No new budget.

## How to Run
pip install flask pandas plotly
python app.py
Open http://localhost:8080

All data from data.novascotia.ca
- LTC Waitlist
- Home Support Waitlist  
- LTC Facilities
- Nursing Home Wait Times
